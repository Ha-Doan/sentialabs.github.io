# sentialabs.com
Sentia Labs website

# Setup
```bash
bundle install
```

# Preview
```bash
bundle exec jekyll serve
```

# Publishing
```bash
# Create a post file
touch _posts/[date]-[title].md
# Write your post
# Commit
git add .
git commit -am "New post!"
# Push
git push origin master
# Reload website
# ...
# Profit
```
